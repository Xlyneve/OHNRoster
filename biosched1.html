Biologics Vaccine Planner

Rule-based planning page for adults preparing to start biologic or other immunosuppressive disease-modifying therapy. It uses built-in vaccine schedules and prioritises live vaccines and longer multi-dose series early so the whole plan can be finished as soon as possible.
● Live vaccine ● Recommended / not funded

Patient + planning setup

Visits are shown from the first visit date, then in 4-week blocks after that.

How it works

Standard schedules are built into the planner for safer planning.

Ticking Dose 1, Dose 2, or Dose 3 means that dose has already been given.
If you know the real date a dose was given, enter it. Later due doses will then use the real date instead of an assumed date.
Planning logic
1. Live vaccines are prioritised early.
2. Longer multi-dose series are also pushed earlier so treatment is not delayed unnecessarily.
3. Standard vaccine spacing is built into the planner logic.
4. The planner tries to bundle vaccines into the fewest visits possible.
5. Each visit is capped at 4 vaccines by default.
6. Pneumococcal logic is linked: Prevenar 13 first, Pneumovax 23 eight weeks later, then boosters five years apart. If Pneumovax 23 is already recorded first, Prevenar 13 is delayed by one year.
